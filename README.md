# _Programming Language Selector_


**By Ryan Walker**

An application designed to help users determine which programming language is right for them. 

## Technologies Used

1. HTML
2. CSS
3. Bootstrap
4. Javascript
5. jQuery
6. VS Code

## Description

This application will determine which programming language is right for users that are interested in learning how to code but may not know where to start. The application will make a suggestion for the language best suited for the user based upon the information gathered during a multiple choice survey. 

## Requirements

* Visit this website: [Project Website](https://ryandanielwalker.github.io/Friday_Number_2)
* Enter your name when prompted.
* Answer all questions truthfully.
* Follow the link provided for more information on the suggested language.
